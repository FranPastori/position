Por padrão ao adiconarmos um elemento HTML ele possui posição static (ele segue o fluxo natural)

Esse elemento tem position static

Nesse caso não temos acessoa as propriedades:

top: Distância do elemento ao topo

left: Distância do elemento a esquerda

right: Distância do elemento a esquerda

buttom: Distância do elemento a base

Ao alterarmos a propriedade position para absolute, relative, fixed e sticky essas propriedades são habilitadas e podemos posicionar o elemento seguindo as seguintes características:

O elemento continua embutido no plano de fundo, e sua posição usa como referencia as laterais da página.
